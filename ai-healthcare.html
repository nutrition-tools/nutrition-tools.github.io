AI活用 × ヘルスケア

医療・ヘルスケアの現場でAIを使う前に、
これだけ決めれば怖くない

管理栄養士 / 三毛猫投資家🐱　｜　読了時間：約5分
「ChatGPTで業務を楽にしたいけど、患者さんの情報を入れていいのか不安」

現場でこういう声をよく聞きます。結論から言うと、使い方さえ決めれば怖くない。ルールを整理するだけで、今日から安全に使い始めることができます。

1.なぜ個人情報をAIに入れてはいけないのか

まず、なぜ「個人情報をAIに入れてはいけない」と言われるのか、具体的に整理します。ChatGPTなどの一般向けAIサービスは、デフォルトでは入力した内容がサービスの改善に使われる可能性があります。

医療・ヘルスケアの現場で特に問題になるのは以下のケースです。

  • 患者名・IDなどの識別情報をそのまま入力する
  • 「Aさん（50代男性）の検査値が〇〇で…」と実名や特定できる情報を含めて質問する
  • 施設・クリニック内の未公開情報（患者数・売上・スタッフ情報）を入力する
  • 第三者提供を前提としていない健康データをクラウドサービスに送信する

個人情報保護法の観点から、これらは本人の同意なしに第三者（AIサービス）に提供したとみなされるリスクがあります。また、万が一情報漏洩が起きた場合、施設としての信頼を大きく損なうことになります。

POINT

怖いのは「AIを使うこと」ではなく、「個人が特定できる情報を入れること」。ここさえ守れば、AIは現場の強力な味方になります。

2.入れていい情報・いけない情報の仕分け方

シンプルな基準は1つ。「その情報だけで特定の人が識別できるか」です。

情報の種類 判断 理由・補足
患者名・生年月日・住所 NG 個人識別情報そのもの
「60代女性、HbA1c 7.2」などの匿名ケース OK 特定できなければ問題なし
「糖尿病患者への食事指導の文章を作って」 OK 個人情報ゼロ、テンプレ作成に最適
施設名＋スタッフ名＋具体的なトラブル内容 NG 施設・個人が特定される可能性あり
「栄養指導でよく聞かれる質問への回答例を作って」 OK 汎用的な業務支援として問題なし
「こういう状況の患者に何を勧めるべきか（匿名化済み）」 OK 思考の壁打ちとして活用できる

迷ったときは「名前と所属を隠しても、誰のことか分かってしまうか？」を自問してみてください。答えがYesなら入力しない、それだけです。

3.現場で今日から使えるAI活用の例3つ

個人情報を一切使わずに、業務を大幅に楽にできる使い方を紹介します。

EXAMPLE 01

指導の定型文・チャット文章の作成

「食後血糖が高い方へのフォローチャットの文章を、励ます雰囲気で3パターン作って」といったプロンプトで、すぐに使える文章の叩き台が出てきます。あとは自分らしく微調整するだけ。

💬 プロンプト例：「食事記録を続けられていない参加者に、責めずにまた記録してもらえるよう促すメッセージを書いて。200字以内で」
EXAMPLE 02

患者向け資料・説明文の作成

「糖質制限について患者さんにわかりやすく説明する1枚ものの資料の文章を作って」のように依頼するだけで、専門用語を噛み砕いた説明文の草案が手に入ります。ゼロから書く時間を大幅に短縮できます。

💬 プロンプト例：「GI値について、医療知識がない60代の方に向けてわかりやすく説明する文章を300字で書いて」
EXAMPLE 03

匿名ケースでの思考整理・壁打ち

「60代女性、血糖値が食後に上がりやすい。昼食はコンビニが多い。何から改善を勧めるべきか」のように、個人を特定しない形で相談すると、栄養学的な整理を手伝ってもらえます。

💬 プロンプト例：「50代男性、BMI28、外食中心で野菜が少ない。血糖値改善のために優先すべき食事の改善ポイントを3つ挙げて」
まとめ

AIに「個人情報を入れない」ルールを1つ守るだけで、日常業務の大半はAIで効率化できます。恐る恐る使うより、使い方のルールを決めてから堂々と使う方が、結果的に安全で生産的です。
